# P2-ABL-001/002: Ablation Controls — Zero vs Mean vs Resample vs Patch

## Claim
Zero and mean ablation produce identical KL effects on Qwen models (0.5B and 1.5B), confirming that replacing activations with zeros is equivalent to replacing with the training-mean activation. Gaussian resample produces 30-60% larger effects. Activation patching (clean-to-corrupt and corrupt-to-clean) produces zero KL, indicating that the patching protocol is not working as intended.

## Result

### Hub layer 2, json_schema task

| Ablation Type | 0.5B KL | 1.5B KL |
|--------------|---------|---------|
| zero | 20.45 | 11.28 |
| mean | 20.45 | 11.28 |
| gaussian_resample | 14.79 | 12.82 |
| patch_clean_to_corrupt | 0.0 | 0.0 |
| patch_corrupt_to_clean | 0.0 | 0.0 |
| random_patch | 12.19 | 18.47 |

### Hub layer 2, factual_recall task

| Ablation Type | 0.5B KL | 1.5B KL |
|--------------|---------|---------|
| zero | 15.18 | 6.63 |
| mean | 15.18 | 6.63 |
| gaussian_resample | 9.27 | 13.24 |
| random_patch | 8.56 | 13.75 |

## Controls
- zero == mean (exact match, confirming distribution-centered representation)
- gaussian_resample: higher variance, 30-60% different from zero
- patching: all zeros — protocol failure, not a real null result

## Interpretation
The zero==mean equivalence is a strong finding: it means the model's default activations at hub layers are approximately zero-centered, so zeroing them is equivalent to replacing with the mean. This validates the Phase 1 zero-ablation methodology. The patching failure (all zeros) is a critical negative result that must be resolved before activation patching claims can be made.

## Limitations
1. Activation patching protocol produced all-zero KL — cannot draw conclusions about causal mediation.
2. Only 1-3 prompts per task per layer — low statistical power.
3. Gaussian resample results are noisier (higher std) than zero/mean.
